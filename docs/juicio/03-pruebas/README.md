# Carpeta 3 – Pruebas

Todos los documentos, comunicaciones y evidencias numerados secuencialmente.

## Estructura

Cada prueba debe tener un identificador único y estar referenciada en la cronología.

### Numeración sugerida

| Tipo | Prefijo |
|------|---------|
| Correos electrónicos | CORREO-XX |
| WhatsApp / Telegram | MSG-XX |
| Facturas | FACT-XX |
| Burofax / cartas certificadas | BUR-XX |
| Fotografías | FOTO-XX |
| Contratos / documentos legales | DOC-XX |
| Grabaciones (si las hay) | AUD-XX |
| Otros | OTRO-XX |

## Para cada prueba, anotar

1. Identificador (ej. CORREO-01)
2. Fecha
3. Emisor y receptor
4. Descripción breve (una línea)
5. Por qué es relevante para la teoría del caso
6. Archivo original (si está digitalizado)
