# Preparación del juicio — FOODAY PROJECT S.L. vs. David Mañas

> Esta estructura es independiente de quién presente la demanda. Sirve para preparar un juicio, lo solicite quien lo solicite.

---

## Estructura

```
docs/juicio/
├── README.md                    ← este archivo
├── teoria-del-caso.md           ← una página: la historia que el juez debe ver
├── 01-contrato/                 ← contrato, anexos, cláusulas clave
│   └── README.md
├── 02-cronologia/               ← línea temporal día por día
│   └── README.md
├── 03-pruebas/                  ← todos los documentos numerados
│   └── README.md
└── 04-debilidades/              ← puntos débiles propios (honestidad brutal)
    └── README.md
```

---

## Cómo usar esta estructura

1. **Rellena `teoria-del-caso.md` primero.** Define la historia en una página. Todo lo demás debe reforzarla.
2. **Alimenta la cronología** con cada hecho, referenciando pruebas.
3. **Numera y archiva las pruebas** en `03-pruebas/`.
4. **Revisa las debilidades** con honestidad antes de cada decisión importante.
5. **El contrato** en `01-contrato/` es la referencia jurídica de fondo.
