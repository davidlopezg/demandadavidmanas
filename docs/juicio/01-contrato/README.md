# Carpeta 1 – Contrato

Copia del contrato de arrendamiento y cualquier anexo, modificación o documento contractual relevante.

## Contenido esperado

- Contrato de arrendamiento original (íntegro).
- Anexos o modificaciones posteriores (si existen).
- Cualquier comunicación que haga referencia a cláusulas concretas.

## Notas

- Tener localizadas las cláusulas clave (obligaciones de las partes, régimen de reparaciones, seguros, suministros, resolución).
- Anotar qué cláusulas podrían ser ambiguas o interpretables en nuestra contra (ver Carpeta 4 – Debilidades).
