# Carpeta 4 – Debilidades

Relación de puntos débiles de nuestra posición que la parte contraria podría explotar.

> **Principio:** cuanto antes los aceptes tú, menos daño harán cuando los use el abogado contrario.

## Formato

| # | Debilidad | Impacto (1-5) | Posible contraargumento o mitigación | ¿Resuelta? |
|---|-----------|---------------|--------------------------------------|------------|
| 1 | Describir el punto débil con honestidad | Gravedad si se explota | Qué podemos decir o hacer para minimizarlo | Sí / No / Parcialmente |

## Instrucciones

- Ser brutalmente honesto. Este documento es para ti y tu abogado, no para el juez.
- No infravalorar ni exagerar. El impacto debe ser realista.
- Si una debilidad ya está resuelta documentalmente, indicarlo y referenciar la prueba.
- Revisar antes de cada decisión importante.
