# Carpeta 2 – Cronología

Línea temporal día por día de todos los hechos relevantes.

## Formato

| Fecha | Hecho | Prueba asociada | Relevancia |
|-------|-------|-----------------|------------|
| dd/mm/aaaa | Descripción objetiva del hecho | Ref. documento (Correo X, WhatsApp Y, Factura Z) | Por qué importa para la teoría del caso |

## Instrucciones

- Ser quirúrgico: solo hechos, sin adjetivos ni interpretaciones.
- Cada entrada debe referenciar una prueba concreta de la Carpeta 3.
- Incluir también los hechos que podrían sernos desfavorables (no ocultarlos).
- Fecha de inicio: desde el primer contacto o incidencia relevante.
- Mantener actualizada si surgen nuevos hechos.
